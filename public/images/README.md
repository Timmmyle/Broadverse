# Thư mục Hình ảnh Trò chơi (Game Assets Folder)

Bạn có thể đặt tất cả hình ảnh/icon cho trò chơi tại đây. Thư mục này được ánh xạ trực tiếp sang đường dẫn `/images/*` trên website.

## Cấu trúc đề xuất:
- `/images/avatars/`: Đặt các ảnh đại diện (avatar) của người chơi.
- `/images/shop/`: Đặt các ảnh cho vật phẩm trong shop (custom symbols, avatar frames, board themes).
- `/images/logo.png`: Ảnh logo của website.

Các định dạng khuyên dùng: `.png`, `.jpg`, `.gif` (hoặc `.svg` cho ảnh pixel sắc nét).
